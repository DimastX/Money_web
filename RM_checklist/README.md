# Redmine Checklist Sorter

Этот проект представляет собой веб-приложение для автоматической сортировки пунктов чеклиста в задачах Redmine. Скрипт упорядочивает пункты по их ID (в порядке возрастания).

## Установка

1. Убедитесь, что установлен Python 3.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настройте переменные окружения:
   - Переименуйте файл `env.example` в `.env`.
   - Откройте `.env` и укажите ваш API ключ Redmine и URL сервера.
     ```ini
     REDMINE_URL=https://redmine.starline.ru
     REDMINE_API_KEY=ващ_api_ключ
     ```

## Использование

1. Запустите приложение:
   ```bash
   python app.py
   ```
2. Откройте браузер по адресу `http://localhost:5000`.
3. Введите номер задачи (ID) и нажмите "Sort Checklist".

## Логика сортировки

Скрипт сортирует пункты чеклиста исключительно по их уникальному идентификатору (`id`) в порядке возрастания. Это соответствует хронологическому порядку создания пунктов (если ID инкрементальные).
